**About**

This repository contains a Node.js Discord bot with modular commands, events and structures. It's organized into folders such as `commands/`, `events/`, `models/`, and `structures/` to keep features separated and maintainable.

**Contents (high level)**

- `commands/` - all bot command modules grouped by category
- `events/` - event handlers for Discord events
- `models/` - data models used by the bot
- `structures/` - helper classes and utilities

---

**Install & Run (quick)**

1. Install Node.js (v18+ recommended).
2. Install dependencies:

```powershell
npm install
```

3. Configure `config.json` with your bot token and settings.

4. Run the bot:

```powershell
node shards.js
```
